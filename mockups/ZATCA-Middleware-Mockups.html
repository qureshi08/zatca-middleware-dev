Set up your accounting software

Do this inside your software. Values below are pre-filled for your account.

  1. Create an OAuth Self-Client in the Zoho API Console; copy Client ID & Secret, generate a refresh token (scope ZohoBooks.fullaccess.all).
  2. Create custom fields on Invoices (Settings → Field Customization):
    cf_zatca_uuid · cf_zatca_status · cf_zatca_qr_code · cf_zatca_error
  3. Create a Workflow webhook (on invoice created/sent):
    https://app.cbt-zatca.com/api/zoho/webhook
    x-api-key: sk_zatca_live_9f2a…c41
  1. Enable the ZATCA fields on account.move (we auto-provision x_zatca_* when you connect, or add manually):
    x_zatca_status · x_zatca_uuid · x_zatca_qr_code · x_zatca_xml · x_zatca_error
  2. Create an Automation / webhook (on invoice posted) calling:
    https://app.cbt-zatca.com/api/odoo/webhook
    x-api-key: sk_zatca_live_9f2a…c41

No software-side setup. You'll call our API directly from your system.
